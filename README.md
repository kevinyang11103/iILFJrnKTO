# 在线电影院系统 python508

## 项目概述

本在线电影院系统基于Python和Django框架开发，结合了Vue.js和MySQL技术，实现了影院信息管理、电影展示、在线购票等核心功能。

## 技术栈

- 后端：Python 3.x + Django
- 前端：Vue.js
- 数据库：MySQL 8
- 开发工具：IDEA + Vscode

## 系统架构

- 架构模式：MVC、前后端分离
- 操作系统：Windows 10 / MacOs

## 功能模块

- 影院信息管理
- 电影展示
- 在线购票
- 用户评论
- 个人中心

## 系统角色

- 管理员：负责管理影院信息、电影数据等
- 用户：浏览电影、购票、评论等

## 运行环境

- 数据库：MySQL 8
- 前端依赖管理：npm
- 数据库可视化工具：Navicat

## 部署步骤

1. 安装Python 3.x和MySQL 8
2. 克隆项目代码到本地
3. 使用IDEA或Vscode打开项目
4. 配置数据库连接
5. 运行前端项目（npm相关命令）
6. 运行后端项目（使用Django命令）
